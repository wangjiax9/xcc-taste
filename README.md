开发工具看着还是蛮舒服的，调试很便利
但是有些地方用起来比较难受  
* 没有创建文件快捷键，创建新页面的时候得右键两级菜单后才OK
* 不能Ctrl+C和Ctrl+V复制文件，右键菜单都没有快捷键
* 环境封闭，能应用的场景相对有限
* 由于我注册的是个人小程序，还无法使用web-view内嵌网页，只能在开发环境中使用
